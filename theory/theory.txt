6. Это пространство внутри grid-контейнера, он захватывает пространство как одной ячейки так и нескольких.
7. .app {
    display: grid;
    grid-template-columns: 2fr 1fr;
    grid-template-rows: 200px 100px;
    grid-template-areas: "section-1 section-1"
        "section-2 section-3";
    gap: 5px;
}

.section-1 {
    grid-area: section-1;
    background-color: blueviolet;
}

.section-2 {
    grid-area: section-2;
    background-color: rgb(43, 226, 171);
}

.section-3 {

    background-color: rgb(226, 43, 195);

8. используя auto-fill

9. Именновынные линии- это линии грида, им дают имена в квадратных скобках [], между для именами указывается ширина столбца или высота строки, которые находятся между этими линиями.
grid-template-columns: [col1start] 1fr [col1end] 10px
                        [col2start] 1fr [col2end] 10px
